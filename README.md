# Extension Security Research PoC

### ⚠️ FOR EDUCATIONAL AND RESEARCH PURPOSES ONLY
This repository contains a Proof-of-Concept (PoC) Chrome Extension designed to demonstrate the security implications of broad extension permissions. 

## Overview
This project illustrates how a browser extension, when granted specific permissions (like `content_scripts`), can interact with page elements and capture input data. This is intended to help developers and security researchers understand "form grabbing" risks and improve defensive coding practices.

## Usage (Research Only)
1. Enable **Developer Mode** in `chrome://extensions`.
2. Use **Load Unpacked** to select this directory.
3. Test only on local or controlled environments.
